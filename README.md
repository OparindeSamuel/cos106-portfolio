# Student Portfolio Website

## Project Overview

This project is a responsive Student Portfolio Website developed as part of the COS 106 Web Technology Lab Assessment. The website showcases my profile, skills, and academic projects using HTML, CSS, and JavaScript.

## Features

- Home Page
- About Me Page
- Projects Page
- Contact Page
- Advanced Calculator
- Student Planner
- Student Result Management System
- Online Quiz System
- Responsive and user-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript
